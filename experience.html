Framatome

May, 2021 - Anticipated May, 2022

Framatome is a global leader in nuclear energy technology. Information about Framatome can be found here. Extensive use of codes are required for analysis of thermal hydrualic events in the reactor core. Starting in May of 2021, I worked in the Core T-H division where engineers study these events. I developed a code called pow3DCreator that generated POWER3D files (a file representing fuel rods in the core). Before pow3DCreator, engineers would create one-time use scripts to transform data, as input from clients is not in the same format. pow3DCreator is capable of transforming multiple client data types into a clean, usable POWER3D file.

  • Lead developer of certified pow3DCreator code for heat flux calculations in the nuclear core.
  • Responsible for codes in development cycle including requirements, testing, and release.
  • Performed user diagnostics to provide nuclear engineers with relevant tools.
